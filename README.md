This project was completed as part of the Quantium Data Analytics Job Simulation on Forage.

The task involved building a simple interactive Dash application to help assess the impact of price changes on sales performance for the Pink Morsel product.

## Overview
The dashboard allows users to:
- View daily sales trends for January 2021 (Can be changed)
- Filter sales data by region using radio buttons
- Interact with a clear and intuitive visual interface

The project also includes a small automated test suite to validate that the application renders correctly.

## Features
- Dash + Plotly interactive line chart
- Region-based filtering
- Simple, user-friendly layout
- Automated UI tests using pytest and Selenium
- Bash script to run tests automatically

## Technologies Used
- Python
- Dash / Plotly
- Pandas
- Pytest
- Bash
